This plugin removes the JetPack plugin for non-admins, and redirects them to `/wp-admin/` if reaching by url.

== Description ==

By installing this plugin will the JetPack plugin automaticly dissappear from users who are not Admins of the website. By using this plugin can you simply add multiple contributors and admins without having to think about them using the plugin.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `/plugin-name/` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Enjoy the plugins functionality
